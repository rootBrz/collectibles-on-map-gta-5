Displays and correctly tracks all collectibles in GTA V on map and radar.  

## Config

```
[Collectibles]
SpaceshipPart=true
LetterScrap=true
NuclearWaste=true
SubmarinePart=true
HiddenPackage=true
MonkeyMosaic=true
PeyotePlant=true
EpsilonTract=true
UnderTheBridge=true
KnifeFlight=true
StuntJump=true

[Keybinds]
; Virtual-key code for toggling all collectibles on/off
; Default: F11
; https://github.com/sepehrsohrabi/Decimal-Virtual-Key-Codes
ToggleKey=122
```
